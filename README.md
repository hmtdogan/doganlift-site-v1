# Proje Kurulum ve Başlatma Rehberi

Bu proje, React bileşenlerini (Hero Odyssey ve Map) içeren bir Next.js uygulamasıdır. Setup işlemi otomatik olarak tamamlanmıştır.

## Nasıl Çalıştırılır?

1.  Terminali açın ve proje klasörüne gidin:
    ```bash
    cd web-app
    ```

2.  Geliştirme sunucusunu başlatın:
    ```bash
    npm run dev
    ```

3.  Tarayıcınızda şu adresi açın:
    `http://localhost:3000`

## İçerik

-   **Hero Odyssey**: Animasyonlu ve etkileşimli hero bölümü.
-   **Harita (Map)**: MapLibreGL tabanlı harita bileşeni.
-   **Shadcn UI**: Modern UI bileşenleri (Button, Card).

## Dosya Yapısı

-   `app/page.tsx`: Demo sayfası.
-   `components/ui/`: Bileşen dosyaları.
-   `components/demo/`: Demo kullanım örnekleri.
